## 프로젝트 소개
SNS와 평정 데이터를 활용한 클래식 영화 백과사전 사이트

<br>

## 사용기술 스택
- Front-end: HTML, SCSS, JAvascrupt, React, React-Router
- Back-end: Phython, Django, MySQL

<br>

## Front-end 필수구현 기능
### 1. 회원가입 / 로그인 페이지
  > - 유효성 검사, back-end API 통신
  - Nav 버튼 클릭 시 로그인 / 회원가입 모달창 오픈

### 2. 메인 페이지
  > - 라이브러리 없이 이미지 슬라이드 구현
    - 3개의 API 한 페이지에 받아오기
